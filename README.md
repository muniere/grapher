# Grapher

Android playground of GitHub API v4 with Kotlin

## Requirements

- Kotlin 1.2.70+
- Android Nougat+
- Android Studio 3.2+

## Get Started

```bash
# Download
$ git clone git@github.com:muniere/grapher.git

# Configuration
$ vim app/src/main/res/values/token.xml

# Open this project with Android Studio, and Select "Run > Run 'app'"
```
